# References

This directory is for source artifacts used to recreate and measure the
Printables model.

Large binary files downloaded from Printables live under:

```text
references/printables-1319709/files/
```

That folder is intentionally ignored by git. Keep generated manifests and notes
under `references/printables-1319709/manifests/` so the local reference cache is
reproducible without committing third-party binary assets.
